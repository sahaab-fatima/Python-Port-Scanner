# 🛡️ Advanced Network Port Scanner

This project is a comprehensive security tool developed to identify open ports on a network. It features both a command-line interface and a visual dashboard for better analysis.

## 🚀 Key Features
- **Fast Port Scanning:** Efficiently scans multiple ports using Python's socket library.
- **Visual Dashboard:** Includes a dedicated script (`dashboard_scanner.py`) for a graphical user interface.
- **Detailed Documentation:** Project architecture and working explained in the provided presentation.

## 📁 Repository Structure
- `my_scanner.py`: Core logic for the network scanner.
- `dashboard_scanner.py`: GUI-based dashboard for scanning.
- `Port Scanners.pptx`: Technical slides and project overview.

## 🛠️ Built With
- **Language:** Python
- **Libraries:** Socket, Tkinter (for dashboard), Datetime
